## About me

I'm a **Computer Science student** who’s been exploring programming through two worlds:  
**university projects** (like a health Android app) and **personal projects** — like game Modding — where I often face heavier logic challenges.

Right now, I’m mostly focused on **Java** (Spring Boot + Android), and I’m especially interested in solving **complex logical problems** — the kind that show up in **automation**, **integrated systems**, and real workflows that need to stay consistent.

My values are simple: **explore, learn constantly, and grow by knowing when to listen**.
